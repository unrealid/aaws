    Setting Logger dengan Aplikasi dEX-2

    • Step 01 - Silahkan download aplikasi dEX-2 melalui link yang telah diberikan pada bagian Software

    • Step 02 - Lalu lakukan install aplikasi tersebut pada PC/Notebook

    • Step 03 - Setting PC/Notebook dengan IP 192.168.10.13 lalu koneksikan logger ke PC/Notebook menggunakan kabel LAN

    • Step 04 - Buka aplikasi dEX. Klik tanda + yang ada pada panel disebelah kiri untuk membuat profile. Isikan Name: AAWS (atau lainnya), Shortname: AAWS (atau boleh dokosongkan), IP Address: 192.168.10.21 (IP Address logger), Port: 80 lalu klik Apply dan tes connectivity

    • Step 05 - Jika connect maka tampilan yang muncul seperti gambar berikut

    Notes

    Kegunaan icon button pada aplikasi dEX-2:

    • Command: Untuk upload program, time, IP Address, dll. Upload script program menggunakan dEX harus dicopy dan dienter per baris

    • Logs: Untuk retrieve program yang di logger. Klik logs > pilih service log > klik service log (jika di logger muncul tanda seru berwarna merah, maka tekan tanda panah ke bawah pada logger). Lalu select baris program dari begin sampai run and jobs, copy ke notepad++ dan save.

    • Download data: pilih menu Data Explorer > Isi range, start and end time > Download
